
# Sorting Algorithm Visualizer

A web-based interactive sorting algorithm visualizer that allows users to see how different sorting algorithms work in real-time.

## Features

- **Multiple Algorithms**: Choose from Bubble Sort, Insertion Sort, Heap Sort, Merge Sort, and Quick Sort
- **Customizable Array Size**: Adjust the array size from 50 to 150 elements using a slider
- **Speed Control**: Select visualization speed (Slow, Medium, Fast)
- **Interactive Controls**:
  - Randomize Array: Generate a new random array
  - Sort: Start the sorting visualization
- **Visual Feedback**: Bars change color to show comparisons and swaps during sorting

## How to Use

1. Open `sortt.html` in a web browser
2. Select your preferred sorting algorithm from the dropdown
3. Adjust the array size using the slider
4. Choose the visualization speed
5. Click "Randomize Array" to generate a new set of bars
6. Click "Sort" to start the visualization

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Project Structure

- `sortt.html`: Main HTML file
- `stylee.css`: Stylesheet for the visualizer
- `indexx.js`: JavaScript logic for sorting algorithms and visualization
